## About

I build AI systems and secure the infrastructure around them.

Not just the model. The full stack: APIs, data pipelines, cloud resources,
containers, secrets management, alerting, audit logs, privacy controls.
Everything the model depends on to actually run in production.

Right now I'm working on a real-time theft detection platform YOLOv8-pose
for pose estimation, LSTM for behavior sequencing, FastAPI backend, MongoDB,
Telegram alerts, Azure Event Hub. The kind of system where a gap in security
isn't a code smell, it's a liability.

My other projects follow the same pattern: RBAC, audit logs, GDPR-style data
rights, rate limiting, signed artifacts, container scanning. Not bolted on
after the fact. Built in from the start.

I learn through building, not tutorials.
