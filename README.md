## About Me

I build **secure AI-powered systems** that connect machine learning, backend engineering, DevOps, and cloud security.

My direction is clear: **Cloud Security for AI Systems**. I focus on protecting the full environment around AI applications: models, APIs, datasets, cloud resources, containers, secrets, alerts, audit logs, and user privacy.

- Building a **real-time AI theft detection platform** with YOLOv8-pose, LSTM, FastAPI, MongoDB, Telegram alerts, Event Hub, and Azure architecture.
- Developing **security-focused portfolio projects** with RBAC, audit logs, GDPR-style data rights, rate limiting, and secure deployment patterns.
- Practicing **DevSecOps, MLOps, cloud infrastructure, and AI security** through real projects instead of isolated tutorials.
- Preparing a long-term path toward **Cloud Security × AI research and engineering**.
